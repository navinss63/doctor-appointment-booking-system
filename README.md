# Doctor Appointment Booking System

## Overview

A backend application developed using Java, Spring Boot, Spring Data JPA, Hibernate, and PostgreSQL for managing doctors and appointment scheduling.

## Features

* Add Doctor
* Book Appointment
* View Appointments
* Cancel Appointment
* Double Booking Prevention
* Date and Time Validation
* Global Exception Handling
* RESTful API Development

## Technologies Used

* Java 17
* Spring Boot
* Spring Data JPA
* Hibernate
* PostgreSQL
* Maven

## Project Architecture

Controller → Service → Repository → Database

## API Endpoints

### Add Doctor

POST /doctors

### Book Appointment

POST /appointment

### View Appointments

GET /appointment

### Cancel Appointment

DELETE /appointment/{id}

## Future Enhancements
